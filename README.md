# Halima Imaxabdi — Portfolio

A simple portfolio website built with TypeScript and Vite. Hosted on Vercel.

Live demo: https://hal-s-portfolio.vercel.app/

## Quick start

```bash
git clone https://github.com/hal-imaxabdi/halima_portfolio.git
cd halima_portfolio
npm install
npm run dev
```

## Build

```bash
npm run build
```
